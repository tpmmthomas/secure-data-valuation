## Environment set up
```bash
conda create -n secdataval python=3.7 -y #Crypten Requires 3.7 
conda activate secdataval
SKLEARN_ALLOW_DEPRECATED_SKLEARN_PACKAGE_INSTALL=True pip install -r requirements.txt
```
